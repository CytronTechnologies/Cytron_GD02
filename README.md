# Cytron_GD02  
G15 Driver (GD02) is used to drive G15 Cube Servo.  

## Requirements  
**Hardware:**   
* [G15 Driver (GD02)](https://my.cytron.io/p-g15-driver)  
* [Cube Servo](https://my.cytron.io/p-cube-servo)  

**Software:**  
* Arduino IDE
